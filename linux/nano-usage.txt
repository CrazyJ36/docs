ARROW           - to set cursor to any starting position for cut/copy
ALT-A			- move to select text 
SHIFT ARROWS	- also selects text
ALT 6           - to copy selected text
CTRL K          - to cut text
CTRL U          - to paste at any position in file
CTRL W			- delete back whole word
CTRL ARROW      - go back, forward one word(left-right) or paragraph(up-down)
